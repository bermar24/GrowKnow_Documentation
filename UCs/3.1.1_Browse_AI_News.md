# 1 Use-Case Name
Browse AI News

## 1.1 Brief Description
A user can browse a feed of AI-related news articles, view summaries and full articles links, search and filter results. 

# 2 Flow of Events
##  2.1 Basic Flow
- User opens the "AI News" section from the main navigation.
- System displays a list of recent AI news items with headline, summary, source, publication date and tags.
- User uses the search box or filters (date range, source, tags) to narrow results.
- User clicks a news item to open the details view showing full text and related links.

### 2.1.3 Narrative
```gherkin
Feature: Browse AI news

  As a user
  I want to browse and search a feed of AI-related news
  So that I can keep up to date with the latest developments.

  Scenario: View recent AI news
    Given I am on the AI News page
    When the page loads
    Then I see a list of recent AI news items with titles and summaries

  Scenario: Open news details
    Given a news item is visible
    When I select the news item
    Then I see the full article and related links
```
This Narrative is implemented in a [seperate document](https://github.com/bermar24/growknow/blob/main/features/steps/browse_ai_news_steps.py).

## 2.2 Alternative Flows
- No results found: system displays a friendly "no articles" message and suggestions to broaden the search.
- Network/API error: system shows an error message with a retry option.
